\subsection{Detailed Description}
cpp\-\_\-lib32 high-\/level events manager. 

Any cpp\-\_\-lib32 event that acts as a wrapper of a system message (Or messages), are managed by this class.

\begin{DoxyRemark}{Remarks}
A \hyperlink{classdl32_event}{dl32\-Event} can wrap more than one system message, that is the reason that the parameter \char`\"{}system\-Messages\char`\"{} is a std\-::vector$<$\-U\-I\-N\-T$>$, not only one U\-I\-N\-T. 

For example\-: System has three different messages (W\-M\-\_\-\-L\-B\-U\-T\-T\-O\-N\-D\-O\-W\-N , W\-M\-\_\-\-R\-B\-U\-T\-T\-O\-N\-D\-O\-W\-N , W\-M\-\_\-\-M\-B\-U\-T\-T\-O\-N\-D\-O\-W\-N) for mouse down events, one per button. User can define a general Mouse\-Down event that catches this three messages.
\end{DoxyRemark}
\begin{DoxyReturn}{Returns}
true if all of the system messages provided are not in use. False in other case. 
\end{DoxyReturn}
