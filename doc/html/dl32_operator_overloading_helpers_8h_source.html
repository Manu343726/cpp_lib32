13 /****************************************************************************
14  * This header provides a set of operator overloading helpers based on CRTP *
15  * *
16  * This helpers provides complete operator overloading based on a specific *
17  * operator that must be implemented by the user of the helper (The class *
18  * that inherits from the helper). *
19  * For example: dl32EqualityHelper provides an implementation of inequality *
20  * ( operator!= ) based on the equality criteria provided by the user *
21  * ( operator== ). *
22  ***************************************************************************/
