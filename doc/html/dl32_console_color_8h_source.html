28 /****************************************************************************************************************************************
29  * This API provides a set of easy-to-use methods for change the style of the standard output stream. *
30  * All operations are performed by the dl32ConsoleColorSettings singleton, but the API provides a more *
31  * confortable syntax to use its methods, all based in a stream syntax: *
32  * *
33  * a) Explicit operations: *
34  * - User specifies the operations in a explicit manner, through a set of types. (See dl32ChangeForegroungColor class, *
35  * dl32ChangeBackgroundColor class , dl32PushStyle class , dl32PopStyle class , and dl32SetAutoPush class bellow) *
36  * - Example: cout << dl32ChangeForegroundColor( dl32ConsoleColor::GREEN ) << "Ohhh, i'm green" << endl; *
37  * *
38  * #Also, the API provides a set of constants to avoid instantations. For example: cout << push_style; instead of *
39  * cout << dl32PushStyle(); *
40  * *
41  * b) Implicit operations: *
42  * - User specifies the style-change using a set of values stored in enums (See dl32StyleChange enum operator<< overload bellow). *
43  * - Example: cout << dl32StyleChange::FOREGROUND << dl32ConsoleColor::GREEN << "Ohhh, i'm green" << endl; *
44  * *
45  * *
46  * An example of use is provided in the console_color_test: *
47  * (https://github.com/Manu343726/cpp_lib32/blob/BigRefactoring/code/tests/refactoring/console_color_test.cpp) *
48  * *
49  ***************************************************************************************************************************************/
